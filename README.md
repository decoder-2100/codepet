## 快速开始

### 系统要求

- Windows 10/11 (x64)
- WebView2（Windows 10 20H2+ 已内置）
- 网络连接（仅 LLM 对话和 AI 分析需要）

### 安装

1. 从 Release 页面下载 `CodePet_0.1.2_x64-setup.exe`
2. 双击安装，完成后自动启动

### API Key 配置（可选）

如需 AI 对话和 Bug 智能分析功能，需要配置 API Key：

1. 右键宠物 → **设置** → **AI 模型** 标签
2. 选择 Provider（如 DeepSeek、通义千问等）
3. 填入 API Key，点击 **测试连接** 验证
4. 点击 **保存**

> 不配置 API Key 时，核心动画、音效、离线吐槽、Bug 粉碎（ASCII 降级）等功能均可正常使用。

## 基本操作

### 窗口布局

宠物默认出现在屏幕右下角，始终置顶。窗口尺寸根据状态自动调整：

| 状态 | 尺寸 | 说明 |
| ---- | ---- | ---- |
| 收起（仅宠物） | 200 × 280 | 只显示宠物画布 |
| 展开（含聊天） | 480 × 680 | 左侧滑出聊天面板 |

### 移动宠物

按住宠物任意位置拖拽，可改变窗口位置。

### 右键菜单

在宠物窗口任意位置右键弹出菜单：

| 菜单项 | 功能 |
|--------|------|
| 💬 聊一聊 Chat | 打开聊天面板，窗口展开 |
| ✨ 夸一夸 Compliment | AI 生成赞美（无 Key 时从预设库随机） |
| 😂 吐个槽 Roast | AI 生成技术吐槽（无 Key 时从预设库随机） |
| 🤣 讲个梗 Joke | 让宠物讲个技术笑话 |
| ⚙️ 设置 Settings | 打开独立设置窗口 |
| ❌ 退出 Quit | 退出应用 |

### 系统托盘

系统托盘中有 CodePet 图标：

- **左键单击**：切换宠物窗口显示/隐藏
- **右键菜单**：Show / Settings / Quit

## 宠物交互

### 宠物状态

宠物根据键盘活动自动切换状态：

| 状态 | 触发条件 | 动画 |
|------|---------|------|
| **待机 (idle)** | KPM ≤ 20 持续约 30 秒 | 自然呼吸、头部微摆、尾巴轻摇 |
| **编码中 (typing)** | KPM > 20 | 编码节奏、身体律动 |
| **说话 (talking)** | AI 回复时 | 说话摇头、嘴巴开合 |
| **开心 (happy)** | Bug 粉碎完成后 / 随机 | 欢快跳跃、疯狂甩尾 |
| **粉碎中 (crushing)** | Bug 粉碎时 | 剧烈震动 → 压扁 |
| **累趴 (collapsed)** | 连续编码 2 小时 | 瘫倒慢呼吸 |
| **慵懒躺平 (lying)** | 随机 idle 子行为 | 偶尔尾巴抽动 |
| **惊喜弹跳 (pet-click)** | 左键点击宠物 | 弹跳 + wiggle 效果 |

### 随机闲时行为

待机状态下，每 8-20 秒随机触发：

| 行为 | 时长 | 附带气泡 |
|------|------|----------|
| look-around（好奇张望） | 2.4s | 30% 概率 |
| stretch（伸懒腰） | 2.8s | 100% 概率 |
| paw-tap（欢快拍爪） | 1.6s | 100% 概率 |
| happy（开心跳跃） | 0.7s | 30% 概率 |

闲时气泡共 18 条短语，如 `"摸鱼 ing..."`、`"该喝水了哦！"`、`"我在看着你哦"` 等。

### 自动眨眼

眼睛每隔 1-5 秒自动眨眼，闭眼 40ms、睁眼 60ms。

### 粒子特效

背景持续生成菱形/爱心/星星/圆形粒子，6 种颜色，500-1300ms 间隔，营造氛围。

### 休息提醒

默认每编码 120 分钟（可调 30-180 分钟），弹出气泡提醒休息 + 提示音效。

### 自动隐藏

开启后，60 秒无键盘活动窗口淡出至 8% 透明度。恢复打字立即恢复。面板打开时保护不隐藏。

## AI 对话

### 支持的 Provider

| Provider | 默认模型 | Base URL |
|----------|----------|----------|
| DeepSeek | deepseek-chat | `https://api.deepseek.com/v1` |
| 通义千问 (Qwen) | qwen-max | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| 智谱 GLM | glm-4-plus | `https://open.bigmodel.cn/api/paas/v4` |
| 百度文心 | ernie-4.0 | `https://qianfan.baidubce.com/v2` |
| OpenAI | gpt-4o | `https://api.openai.com/v1` |
| 自定义 (OpenAI 兼容) | — | 可自由配置 |

### 聊天面板

1. 右键 → **聊一聊** 或左键点击宠物打开
2. 输入消息，**Enter** 发送，**Shift + Enter** 换行
3. AI 回复逐 token 流式显示（带闪烁光标指示器）
4. 回复完成后自动存入对话历史

### 欢迎页

无消息时显示 4 个建议芯片：Bug 分析、代码审查、技术段子、休息提醒。

### 会话管理

- 支持多会话：新建、切换、删除
- 历史侧栏按时间倒序展示会话
- 会话标题自动提取自首条用户消息
- 持久化：localStorage + 后端文件系统同步

### Markdown 渲染

宠物回复支持：标题 (h1-h3)、加粗、行内代码、代码块（带语言标签）、无序列表、有序列表。

### 技能面板

聊天面板底部 6 个快捷按钮，点击自动发送预设 Prompt：Bug 分析 / 代码审查 / 技术闲聊 / 情绪调节 / 吐槽 / 提醒。

### 停止生成

流式输出过程中输入框和发送按钮自动禁用，完成或出错后恢复。

## Bug 粉碎

### 使用方式

1. 复制报错信息到剪贴板
2. 将文本**拖拽**到宠物窗口
3. 宠物区域显示红色虚线边框
4. 释放后执行粉碎动画

### AI 分析模式（有 API Key）

1. 执行 crushing 动画（震动 → 压扁）
2. 报错发送给 LLM 分析
3. 1 秒后展示分析结果气泡
4. 切换为 happy 动画

### 离线降级模式（无 API Key）

弹出 ASCII 艺术风格的粉碎报告：

```
╔══════════════════════╗
║   🐛 Bug 粉碎报告      ║
╠══════════════════════╣
║ "TypeError: Cannot..."  ║
║                       ║
║ 状态: ✅ 已粉碎       ║
║ 建议: 重启试试        ║
╚══════════════════════╝
```

## 键盘活动监测

### 监测方式

后台线程使用 `rdev` 全局键盘钩子，每 5 秒计算一次 KPM（每分钟按键数）。

### 状态切换

- **编码状态**：KPM > 20 → typing 动画
- **待机状态**：KPM ≤ 20 持续约 30 秒 → idle 动画
- **连续编码提醒**：累计 2 小时 → collapsed 状态 + 休息提醒

## 常见问题

**Q: 退出后设置会丢失吗？**
A: 不会。设置保存在可执行文件同目录下的 `coderpet_settings.json` 中。

**Q: 如何更新 API Key？**
A: 右键 → 设置 → AI 模型 标签，修改后点击保存。

**Q: 宠物窗口消失了怎么办？**
A: 点击系统托盘中的 CodePet 图标，或右键托盘图标选择 "Show"。

**Q: 可以不配置 API Key 使用吗？**
A: 可以。核心动画、宠物自定义、离线吐槽、Bug 粉碎（降级模式）均无需 API Key。

**Q: 如何修改宠物的性格？**
A: 右键 → 设置 → 宠物角色 → 性格下拉框 → 保存。

**Q: 自定义的外观会保存吗？**
A: 自定义面板中修改后点击 **保存** 按钮持久化到磁盘。

**Q: CPU 占用高吗？**
A: 空闲时 CPU < 5%，Canvas 渲染 60fps，时间步长上限 50ms。

**Q: 安装文件有多大？**
A: NSIS 安装包约 4MB，静态链接 MSVCRT，无需额外安装运行库。

### 构建

```bash
# 开发模式（Vite dev server + Tauri window）
npm run tauri dev

# 前端-only dev（无 Tauri 后端）
npm run dev

# 类型检查
npx tsc --noEmit

# 生产构建（便携 exe + NSIS 安装器，静态 CRT 链接）
cd src-tauri
RUSTFLAGS="-C target-feature=+crt-static" cargo tauri build --bundles nsis
# 输出：src-tauri/target/release/bundle/nsis/CodePet_<version>_x64-setup.exe

# Rust 测试
cd src-tauri && cargo test

# 前端测试
npm test
npm run test:watch
```

### 设置持久化

`serde(alias)` 支持 snake_case/camelCase 字段互转，新增字段通过 `serde(default)` 向后兼容。
