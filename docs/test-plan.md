# CodePet 测试计划

## 测试范围

覆盖需求文档中定义的 8 个功能模块（F1-F8）共 39 项功能，以及 7 项非功能需求（NFR1-NFR7）。

## 测试环境

| 项目 | 配置 |
|------|------|
| OS | Windows 11 / macOS / Linux |
| 屏幕 | 支持 HiDPI，最低 1280×720 |
| 网络 | LLM 相关测试需要网络连接 |
| API Key | DeepSeek / Qwen / OpenAI 任一组有效的 API Key |
| Node.js | >= 18 |
| Rust | >= 1.70 |
